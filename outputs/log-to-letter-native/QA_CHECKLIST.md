# Log Planet QA 체크리스트

출시 전 QA는 이 문서를 위에서부터 순서대로 진행한다.

상태 표기:

- `[ ]` 미확인
- `[x]` 통과
- `[!]` 이슈 있음
- `[-]` 이번 빌드에서 해당 없음

## 테스트 환경

- iOS 시뮬레이터: 작은 화면 1종, Dynamic Island 화면 1종
- 실제 iPhone 또는 TestFlight: 1종 이상
- 비로그인 상태와 로그인 상태 각각 1회
- Wi-Fi 연결, 오프라인, 재연결 상태 각각 확인
- 신규 설치와 기존 데이터 보유 상태 각각 확인
