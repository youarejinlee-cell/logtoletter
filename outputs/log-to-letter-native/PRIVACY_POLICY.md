# Log Planet 개인정보 처리방침

시행일: 2026년 7월 18일

`Log Planet`은 사용자가 순간의 생각과 감정을 기록하고, 월별 기록을 행성과 분석으로 돌아볼 수 있도록 돕는 앱입니다. 이 문서는 앱이 어떤 정보를 처리하는지, 어떤 목적으로 사용하는지, 사용자가 자신의 데이터를 어떻게 관리할 수 있는지 설명합니다.

## 1. 처리하는 개인정보

앱은 사용 방식에 따라 아래 정보를 처리할 수 있습니다.

### 사용자가 직접 입력하는 정보

- 기록 내용과 기록 카테고리
- 선택한 감정 태그와 에너지 수준
- 사용자가 월별로 작성하고 저장·수정하는 메모
- 알림 사용 여부, 알림 시간, 반복 간격, 요일 등 알림 설정값
- 테마, 에너지 표시 방식, 목표 감정 등 앱 설정값

### 로그인 시 처리되는 정보

Apple, 카카오 또는 Google 계정으로 로그인하는 경우, 로그인 제공자가 사용자 동의에 따라 제공하는 범위에서 아래 정보가 처리될 수 있습니다.

- 이메일 주소
- 이름 또는 표시 이름
- 프로필 이미지 URL
- 로그인 제공자의 사용자 식별자와 Supabase 인증 사용자 ID

로그인 제공자가 이메일, 이름 또는 프로필 이미지를 제공하지 않는 경우 해당 정보는 수집하지 않습니다.

### 기기 권한과 관련된 정보

앱은 아래 기능을 제공하기 위해 사용자의 권한을 요청할 수 있습니다.

- 알림 권한: 기록 알림을 보내기 위해 사용합니다.
- 사진 보관함 추가 권한: 기록 행성을 이미지로 저장하기 위해 사용합니다. 앱은 사진 보관함의 기존 사진을 읽지 않습니다.

행성 탭의 배경음악은 앱에 포함된 오디오 파일을 재생하며, 앱은 마이크 또는 음성 녹음 권한을 요청하거나 사용하지 않습니다.

앱은 사용자가 권한을 허용하지 않아도 가능한 범위 내에서 기본 기능을 제공합니다.

## 2. 개인정보 처리 목적

앱은 개인정보를 아래 목적으로 사용합니다.

- 사용자가 남긴 기록을 저장하고 다시 보여주기 위해
- 캘린더에서 날짜별 기록, 감정, 에너지 흐름을 보여주기 위해
- 월별 기록을 바탕으로 기록 행성과 분석을 제공하기 위해
- 사용자가 월별 메모를 저장하고 다시 확인하거나 수정할 수 있도록 하기 위해
- 사용자가 설정한 알림 방식에 따라 기록 알림을 보내기 위해
- 로그인한 사용자의 기록과 설정을 계정에 동기화하기 위해
- 행성 이미지를 사용자의 사진 보관함에 저장하기 위해
- 사용자의 데이터 및 계정 삭제 요청을 처리하기 위해

## 3. 저장 방식

앱은 로그인 여부에 따라 데이터를 다르게 저장합니다.

### 비로그인 상태

비로그인 상태에서 작성한 기록, 월별 메모와 설정은 사용자의 기기 안에 저장됩니다. 이 경우 서버 계정과 연결되지 않으며, 앱 삭제 또는 기기 저장소 초기화 시 데이터가 사라질 수 있습니다.

### 로그인 상태

사용자가 Apple, 카카오 또는 Google 계정으로 로그인하면 기록, 월별 메모, 알림 설정, 앱 설정 일부가 Supabase 서버에 동기화될 수 있습니다. 이 기능은 여러 기기에서 데이터를 이어서 사용하거나 앱을 다시 설치했을 때 데이터를 불러오기 위해 사용됩니다.

비로그인 상태에서 작성한 기록은 사용자가 로그인할 때 선택한 경우에만 해당 계정으로 가져와 동기화합니다.

## 4. 제3자 서비스

앱은 아래 서비스를 사용할 수 있습니다.

### Google

Google 계정 로그인을 위해 사용합니다. Google 로그인 과정에서 Google이 처리하는 정보는 Google의 개인정보 처리방침을 따릅니다.

### Apple

Apple 계정 로그인을 위해 사용합니다. Apple 로그인 과정에서 Apple이 처리하는 정보는 Apple의 개인정보 처리방침을 따릅니다.

### Kakao

카카오 계정 로그인을 위해 사용합니다. 카카오 로그인 과정에서 Kakao가 처리하는 정보는 Kakao의 개인정보 처리방침을 따릅니다.

### Supabase

사용자 인증, 데이터 저장, 서버 동기화를 위해 사용합니다. 로그인한 사용자의 기록, 월별 메모, 알림 설정과 앱 설정 일부가 Supabase에 저장될 수 있습니다.

### Apple / Expo / EAS

앱 빌드, 배포 및 실행 환경 제공을 위해 사용합니다. 사용자가 작성한 기록 내용이나 월별 메모를 이들 서비스에 앱 기능 제공 목적으로 전송하지 않습니다.

## 9. 보안

앱은 사용자의 기록과 계정 정보를 보호하기 위해 필요한 기술적 조치를 적용합니다.
